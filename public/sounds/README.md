# 音效文件说明

## 📁 目录结构

此目录用于存放应用的音效文件。

## 🎵 需要的音效文件

请准备以下音效文件（MP3格式）：

1. **task-complete.mp3** - 任务完成音效
   - 建议：愉悦的上升音调或"叮"的声音
   - 时长：0.3-0.5秒

2. **task-delete.mp3** - 任务删除音效
   - 建议：简短的下降音或"咻"的声音
   - 时长：0.2-0.3秒

3. **level-up.mp3** - 升级音效
   - 建议：庆祝的音阶或"升级"提示音
   - 时长：0.5-1.0秒

4. **achievement-unlock.mp3** - 成就解锁音效
   - 建议：华丽的和弦或"成就达成"音效
   - 时长：0.6-1.0秒

5. **reward-redeem.mp3** - 奖励兑换音效
   - 建议：金币音效或"购买成功"提示
   - 时长：0.3-0.5秒

6. **button-click.mp3** - 按钮点击音效
   - 建议：短促的咔哒声或轻柔的点击音
   - 时长：0.05-0.1秒

7. **success.mp3** - 成功提示音效
   - 建议：简短的正面提示音
   - 时长：0.2-0.3秒

8. **error.mp3** - 错误提示音效
   - 建议：温和的错误提示音（不要太吓人）
   - 时长：0.2-0.3秒

## 🔊 音效要求

- **格式**：MP3（推荐）或OGG
- **采样率**：44.1kHz或48kHz
- **比特率**：128kbps-192kbps
- **声道**：单声道或立体声
- **音量**：适中，统一归一化

## ⚙️ 临时方案

目前应用使用 **Web Audio API** 生成简单的提示音作为临时方案。
如果音效文件缺失，系统会自动使用程序化生成的音效。

要使用真实音效文件，只需将上述文件放入此目录即可。
